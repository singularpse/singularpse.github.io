Macbook Pro 2016 入手

发表于 2016-12-31

Context

现在实验室还是家里都是连接24寸显示器的，但每天背2012款MBP回家还是觉得好重，尽管2012款光驱位放在旧硬盘，原硬盘位装上SSD，加装8G内存，平常使用速度也没什么抱怨的。键盘是HHKB Pro 2，触控板是Magic TrackPad 2，所以平常笔记本都是合盖连接显示器。音频方面，在家是蓝牙连接Bose Soundlink mini 或者 beyerdynamic DT990 Pro 耳机；实验室则是IE80听音乐，Bose QC30降噪专注。
基于以上个人环境考虑，最亮眼的Touch Bar几乎没有使用场景可言，日本Apple店学生优惠也要18万多，相比而言还是14万的非Touch Bar版对我而言就足够了。

Update


终于等到MacBook Pro大更新，外形和预想中一样，Air和Pro的融合版。变轻变薄变窄，加了深空灰，就这些改变已经足够吸引我了。
重量已经很满意了，虽然依旧被初见时夸赞比她的MBP2015小好多的同学拿起瞬间吐槽比想象中重诶，于是试着计算了一下，的确发现MacBook Pro 系列密确实在不断增加，人类的感受器真是灵敏：

P3色域的屏幕基本没感觉什么区别，毕竟使用场景而言接触的大多都是基于sRGB色域标准设计，目前还是很难感受到太大区别。

扬声器的提升反而是最让人惊讶的，虽然对我而言没什么用。最大音量提升，低频更饱满，最简单也最有效，立刻能感受到区别，算是笔记本电脑里非常出色的外放体验了。
键盘说是手感提升了，但那是相对于NMB，2012款MBP反而是最好MBP系列最好手感，这点也是为了轻薄化的无奈妥协，但基本外接HHKB，很少用到所以对我没太大影响。

左边两个thunderbolt3，因为使用USB type C接口，我倒是没觉得特别不方便，连接一条线就能同时充电，视频输出，USB hub搭配Dell U2417H显示器反而挺方便的。未来Type C设备会慢慢普及，有种言论是买2016款就是当小白鼠，未免有些过激了，实际用下来真没有特别不便的地方。
用了快一个月，至今未用到那个USB C to USB A的转接头，再次为Dell U2417H把两个USB口移到显示器侧边点赞！

精灵宝可梦历代种族值分布

发表于 2016-10-21

昨晚看到新公布的600准神杖尾鱗甲龍以及低配版的老创銀伴戰獸，总体感觉低配老创还是挺有范儿的，种族值估计不会超600，搞不好直接打个八折，720*0.8=576也还挺有趣。

既然提到种族值了，就突然对历代新Pokemon的种族值分布有了点兴趣，Wiki虽然有种族值列表，但一图胜千言，正好最近看到一个很漂亮的ggplot2主题，打开RStudio动手：

  • 虚线标注的是御三家平均种族值530
  • 种族值密度曲线可以看出Pokemon时代的发展演化
  • 红绿时代奠定的基本分布雏形
  • 金银时代新增的100只Pokemon更多作为初代的一种补充，较难独立撑起一个地区的Pokemon生态，所以金银地图里仍然可以见到很多熟悉的初代Pokemon。
  • 宝石时代明确定位上全新地区的概念，新增Pokemon开始能够独立构筑一个全新的生态系统，地区的家门鸟，家门狗等开始有了一个系统式的规范。
  • DP时代高耸的右峰应该很好解释，DP时代是一个造神的时代，众多的分工明确，充满宗教意味的神兽群奠定了Pokemon的世界观，当年玩DP时对老创没什么好感，现在回首望来，阿尔宙斯创造了掌管时间和空间的两位主神，还有PT里掌管反转世界的冥王龙，三位一体的蘑菇（由克希、艾姆利多和亚克诺姆）也代表了知识，情感和意志，据说是来自日本神话的三神器八咫镜（知识），八尺琼勾玉（情感），和天丛云剑（意志），甚至连梦境都有了美梦神和设计最酷的噩梦神，技能物理攻击和特殊攻击的分化等等新系统的导入，可以说DP使得Pokemon这个幻想世界变得自洽，使玩家在幻想世界寻找一种真实感变得不再是一件傻事。
  • BW时代则是一个系统完成度达到某种临界后的的自我革新，彻底脱离日本的原型设计，全地图的PM生态位全部由本地区Pokemon来构建，虽然Pokemon的形象设计槽点无数，都不知从哪里开始黑起…种族值的分布果然完全再现的初代的形状，只是为了给新Pokemon登场理由，这时代的PM种族值都挺暴力的。
  • 终于来到了xy时代，新增PM是最接近初代的分布，或许设计人员本身就是有意为之亦或是多年自然选择的结果？当然本次数据是故意剔除mega和原始回归的Pokemon数据的，这些数据留着有空下次再谈。
  • 马上还有一个多月就要发售的日月，不知又会是什么样的分布，敬请期待吧。

最后，附加一份详细的种族值总和及HP，攻击，防御，特攻，特防，速度的历代分布对比图。

游戏王OCG历史上的主流卡组

发表于 2016-10-11

本文翻译自yugioh-wiki，文中涉及关键词将以斜体标明，OCG卡名将会以加粗标明，卡名取用广泛流通的中文翻译，不熟悉的卡片可以方便地进行搜索其详情。

本项目纯属个人兴趣，不定期更新，请理解。

第1期

本时代尚不存在限制卡和准限制卡，大多是通过强欲之壶和天使的施舍进行大量抽卡，赚取卡差，直截了当地获得胜利。接着出现了将怪兽送往墓地，再通过死者苏生进行特殊召唤的战术，之后又演变出了利用雷击破坏对方怪兽进行攻击的除去后攻击战术。

在第1期中，首次出现了效果怪兽和反击陷阱，决斗的可玩性得到大幅度的拓展。
另外，后期EX包的发售，让恶魔的召唤和死者苏生等当时的必须卡，比较简单就能入手。

关键卡
  • 双生妖精：初代的王牌攻击手。 （译者注：等级4 攻击力1900 通常怪兽）
  • 机械猎手：当时攻击力仅次于双生妖精的机械族怪兽。
  • 恶魔的召唤：当时最具代表性的上级怪兽。（译者注：『我召唤恶魔的召唤！』台词好别扭）
  • 三眼怪：高速检索被封印的艾克佐迪亚系列部件，获取规则胜利变得可能。
  • 黑森林女巫：除了同上用法外，还能对恶魔的召唤等进行检索。
  • 雷击：简单粗暴的最强怪兽除去魔法。
  • 黑洞：同上，除去双方场上所有怪兽。
  • 死者苏生：无Cost的最高苏生卡。
  • 强欲之壶：最高的抽卡辅助卡。（译者注：动画效果的天降宝牌才是最强好不好！）
  • 天使的施舍：同上，另外舍弃卡作为效果，而非Cost，后来诞生了很多相关Combo。
  • 心变：无Cost夺取对方怪兽控制权的最高NTR卡。
  • 鹰身女妖的羽毛扫：最强的魔法陷阱除去卡，当时因为价格高昂，持有该卡的玩家很少。
  • 死之卡组破坏病毒：高价格也高威力。
  • 神圣防护罩-反射镜力-：高泛用性的怪兽除去陷阱卡。
  • 被封印的艾克佐迪亚 系列部件：现在集齐五部件获得胜利的战术依然存在。
主流卡组
  • 标准卡组
  • 被封印的艾克佐迪亚卡组
主力怪兽攻击力的变化

  • 下级怪兽：1200~1900

    最初期限定的以水魔道士（（译者注：等级4 防御力2000 通常怪兽））为主力。而后随着新卡包的发售，下级怪兽攻击力不断上升，最终以双生妖精的1900达到顶峰。

  • 上级怪兽：1700~2500

    暗杀者和魔加农，当时此类1700的通常怪兽已属优秀，诅咒之龙（右键搜索卡图，你一定会很怀念！）则是其中屈指可数的强者了。之后于Vol.4登场的恶魔的召唤作为上级怪兽的代表，很长一段时间内占据着攻击力No.1的宝座。

第2期

本期开始连锁的概念诞生，规则也进行了相应的调整。

速攻魔法也首次出现。

魔法陷阱卡种类增加了标识符，永续魔法等魔法陷阱的种类变得容易分辨。

卡通怪兽和灵魂怪兽命名的拥有部分共同效果的效果怪兽群首次登场。

卡组大多是标准卡组或以杀手番茄为代表的，配上一些强力的限制卡构成的卡组。
终盘时八汰鸟和噩梦的海市蜃楼等强力卡的加入，无可动摇的强大效果也注定被关小黑屋。
这其中利用生还的宝牌为引擎有机会一回合击杀的卡组也引起了极大关注。参见：生还的宝牌-艾克佐迪亚

另外，本期还诞生了四种STRUCTURE DECK，使得雷击和天使的施舍等必备卡入手变得更加简单。

关键卡
  • 人造人-念力震慑者：强大的陷阱封杀效果和上级怪兽里首屈一指的攻击力的决胜卡。
  • 血腥魔兽人：和双生妖精作为当时两大攻击手。
  • 恶魔科学怪人：青眼究极龙和巨大化的配合实现一回合击杀。（译者注：玩过GBA上游戏王6都应该都被社长这招虐过把。）
  • 电子壶：强大的清场重置效果，以此诞生很各式各样的Combo。壶家族无弱手！
  • 纤维壶：强大的重置效果，实在逆天。（反转：双方把自己场上的卡，手牌，墓地的卡和卡组合在一起洗切。之后各自从卡组抽5张卡。）
  • 杀人蛇：当时最高的手牌Cost。（自己的准备阶段墓地有「杀人蛇」存在的场合，可以把「杀人蛇」放回到手卡。）
  • 流氓佣兵部队：当时环境下非常泛用的怪兽除去系卡，被很多卡组采用。
  • 八汰鸟：八汰鸟Lock，坠入地狱般的体验，游戏王史上留名的凶恶卡片。（效果·灵魂：这张卡不能特殊召唤。召唤·反转的回合的结束阶段时回到主人的手卡。这张卡给予对方玩家战斗伤害的场合，跳过下次的对方的回合的抽卡阶段。）
  • 抢夺：心变类似的NTR卡。
  • 强引的番兵：手牌破坏三神器其一。（确认对方的手卡，从那之中把1张卡回到卡组。）
  • 收押：手牌破坏三神器其二。（支付1000基本分发动。确认对方的手卡，从那之中丢弃1张卡。）
  • 爱恶作剧的双子恶魔：手牌破坏三神器其二。（支付1000基本分发动。对方手卡随机1张丢弃，再选择1张丢弃。）
  • 苦涩的抉择：配合补充要员使用，被封印的艾克佐迪亚异常凶恶。（从自己卡组选择5张卡给对方观看。对方从那之中选择1张。对方选择的1张卡加入自己手卡，剩下的卡丢弃去墓地。）
  • 噩梦的海市蜃楼：无论是组Combo还是抽卡辅助，都非常强力。（对方的准备阶段时，自己抽卡直到4张手卡。自己的准备阶段时，随机丢弃那个效果所抽的卡的数量的手卡。）
  • 王宫的敕命：当时兴起的几张有用魔法无效化效果，同时又能自由选择解除时机的卡片。（只要这张卡在场上存在，场上的魔法卡的效果无效。这张卡的控制者在每次自己的准备阶段支付700基本分。或者不支付700基本分让这张卡破坏。）
  • 破坏轮：最高泛用性的凶恶陷阱。

主力怪兽攻击力的变化

  • 下级怪兽：1400or1900~

    前者所指杀手番茄为代表的各类遗言怪，黑森林女巫和三眼怪也和遗言众作为卡组的基本构成。强制转移和八汰鸟的登场，这打法真是越来越无言以对。

    后者所指双生妖精和血腥魔兽人的黄金时代。

  • 上级怪兽：1700~2500

    前期是恶魔的召唤和电子科技翼龙的时代。人造人-念力震慑者登场以后，恶魔的召唤正式退役，其在首位称霸了很长的一段时间。

    后期冥界的魔王 哈·迪斯，吸血鬼领主，天空骑士 珀耳修斯，守护者斯芬克斯等优秀的上级怪兽大量登场。这其中吸血鬼领主因为入手简单，效果也很好用，成为了当时的强力卡。

    也是从这时期开始，上级怪兽揭去了简单粗暴的标签，选择不同的上级怪兽开始成为一种卡组构筑的乐趣。
