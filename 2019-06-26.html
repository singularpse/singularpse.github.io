#
高齢化社会が進む中、老化と生活習慣病によって認知症に罹るリスクは高まっている、申請者はモデル動物としてゼブラフィッシュを用いて加齢性認知機能低下機構の解明と脳機能障害に関する疾病の克服を研究目的とする。

研究の成果’効果

  • 自動化加齢性認知症記憶アッセイ装置とプロトコールを作製、英語原著論文発表、商品化され三重大学に投入され。
  • ゼブラフィッシュの酸素代謝評価システムを構築して、新たな加齢変化基準が確立できた。（英語原著論文Revise中）
  • 糖尿病認知症モデル動物を作製し、定時定量給餌自動化システムを構築した。今年、企業（名東水園）と協同開発して商品発表、特許出願中。（第3回ゼブラフィッシュ創薬研究会発表、英語原著論文準備中）
  • ソフトウェア企業（中立電機）と協同研究して、３Ｄ行動データ取得と機械学習による特徴行動探索システムの開発を行っている。（第23,24回小型魚類研究会発表）

以上の成果とこれからの研究を通し、人の加齢性認知症、糖尿病についての研究と治療に貢献できると考えている。

今後の研究計画’予定

加齢変化因子と新規行動特徴量計測による記憶テストを統合解析し、認知症による「脳認知機能低下」を理解するため、下記四つの研究目標を定める。

  • 目標1. 加齢変化因子測定による老化評価基準化 (①月齢②体長③体重④行動量⑤基礎酸素代謝量⑥血管ネットワーク)
  • 目標2. 認知症ゼブラフィッシュモデルの作製と加齢変化因子測定(①糖尿病認知症モデル②CADASIL遺伝性認知症モデル)
  • 目標3. 認知症モデルを用いて人工知能（AI）で脳機能変化で生じる行動の新規特徴量の探索
  • 目標4. 新規行動特徴量計測による認知能力定量化と加齢変化因子量の認知症統合解析
    • 認知能力と特異的に相関する行動特徴を抽出する
    • 新しく同定した新規特徴量を指標として、Arduino、R言語など自動化解析技術を含めて、より精確な自動認知能力評価システムを構築する
    • 定量化した認知能力情報と本計画で計測する加齢変化因子量の追跡データベースを利用して人工知能（AI）による認知症統合解析を行う。
